## Development

```sh
# Run plugin on Chrome
$ pnpm dev
# or
$ pnpm dev:firefox

# Build script
$ pnpm build
# or
$ pnpm build:firefox

# Zip
$ pnpm zip
# or
$ pnpm zip:firefox
```

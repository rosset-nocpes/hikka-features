Plugin uses:

- [WXT](https://wxt.dev)
- [SolidJS](https://docs.solidjs.com/)
- [TailwindCSS](https://tailwindcss.com/)
- [SolidUI (shadcn-ui fork)](https://www.solid-ui.com/)

## Development

```sh
# Run plugin on Chrome
$ pnpm dev
# or
$ pnpm dev:firefox

# Build script
$ pnpm build
# or
$ pnpm build:firefox

# Zip
$ pnpm zip
# or
$ pnpm zip:firefox
```
